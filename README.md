# Миний анхны төсөл
Энэ бол миний Программ хангамж хөгжүүлэлтийн процесс хичээлийн анхны төсөл.

## Багийн гишүүд
- Бат (Product Owner)
- Сараа (Scrum Master)
- Номин (Хөгжүүлэгч)
- Тэмүүлэн (Хөгжүүлэгч)

## Төслийн зорилго
Энэ төслөөр бид Scrum аргачлалыг дагаж, жижиг веб аппликейшн хөгжүүлэх болно.
